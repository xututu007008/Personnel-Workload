# 机关工作量摸排报告规则

## 一、规则定位

本文件用于指导生成、检查和修改 `机关工作量摸排报告`。本报告是部门工作量摸排、员工评价完成情况、等级结构、岗位必要性、低负荷信号、异常复核和分层管理建议的主报告。

## 二、必须读取的输入

- `职能部门工作量摸排表*.xlsx`
- `部门成员工作主管评定*.xlsx`
- `机关员工主管评价20题重算报告`
- `机关员工工作负荷认定规则*.md`

## 三、评级口径

本报告只允许使用以下三类评级：

| 评级 | 说明 |
| --- | --- |
| 表内自动生成评级 | 原主管评价表自动计算结果 |
| 主管主观评级 | 主管人工给出的评级 |
| 20题规则重算评级 | 按规则重新赋分后的等级 |

`主管评级对应公司等级` 不参与任何分析，不得用于薪酬区间一致性判断。

## 四、必须包含的章节

1. 统计口径
2. 部门与员工数量
3. 各部门工作摸排填报情况
4. 未填报工作摸排人员名单
5. 各部门员工评价完成情况
6. 未完成员工评价人员名单
7. 工作负荷与岗位状态
8. 本维度结论
9. ABCD 等级分布情况
10. 自动评级总体分布
11. 主管主观评级总体分布
12. 自动评级与主管主观评级一致性
13. 自动评级与主管主观评级不一致人员名单
14. 20题规则重算评级总体分布
15. 20题规则重算评级与主管主观评级一致性
16. 20题规则重算评级与主管主观评级不一致人员名单
17. 各部门主管主观评级结构
18. 主管主观评级与20题重算评级一致性
19. 岗位必要性、离职影响与评级反差
20. 各部门等级结构是否符合常态分布
21. 取低档后的异常复核名单
22. 低负荷信号与等级交叉复核范围
23. 特殊人群与分层管理规则
24. 综合结论与建议

## 五、关键计算

1. 填报率 = 已完成工作摸排人数 / 当前范围有效员工数。
2. 评价完成率 = 已完成主管评价人数 / 当前范围有效员工数。
3. 取低档后异常复核：比较自动生成评级和主管主观评级，取较低档作为保守评级，再叠加关键题异常。
4. 主管主观评级与20题重算评级一致性：以 20 题重算评级为基准，与主管主观评级比较高低。
5. 薪酬区间字段只作为主管主观评级对应区间展示，不用于决定等级。

### 异常复核名单

| 列名 |
| --- |

| 主管主观评级 |
| 最终评价级别 |
| 异常项数 |
| 异常选项 |

`异常选项` 必须为最宽列。

## 七、PDF 排版

1. 表头浅蓝底色。
2. 宽表左右页边距不得过小。
3. 等级、人数、姓名列尽量窄，说明列和异常选项列加宽。
4. 行高适中，长文本允许换行。

## 八、验收清单

- 未填报人员名单显示具体姓名。
- 未完成主管评价人员名单显示具体姓名。
- 所有必保章节存在。
- 不出现基于 `主管评级对应公司等级` 的分析。
- 当前范围描述与单部门/部分部门/全部部门一致。

## 九、逐章节标准

| 章节 | 统计项 | 输出标准 |
| --- | --- | --- |
| 部门与员工数量 | 部门数、员工数 | 当前范围单独识别，不得写死全机关人数。 |
| 工作摸排填报情况 | 应填、已填、未填、填报率 | 未填报人员必须列名单。 |
| 员工评价完成情况 | 应评、已评、未评、完成率 | 未完成评价人员必须列名单。 |
| 工作负荷与岗位状态 | 自动负荷、员工态度、主管忙闲 | 每个小节后保留本维度结论。 |
| ABCD等级分布 | 自动、主管、20题重算 | 三类评级分别统计并比较。 |
| 一致性 | 自动 vs 主管、20题重算 vs 主管 | 一致和不一致均需给人数。 |
| 岗位必要性 | 岗位必要性偏弱、离职影响偏弱 | 与评级交叉后列异常样本。 |
| 常态分布 | 各部门等级结构 | 小样本部门需提示样本限制。 |
| 取低档异常 | B类及以上异常、C类异常 | 异常项数和异常选项必须列出。 |
| 低负荷交叉 | 低负荷 + 等级 | 识别需要复核但不直接定档。 |
| 特殊人群 | E类、毕业两年内、顾问转岗年龄 | 按规则保护或分层管理。 |

## 十、异常项标准

异常项至少包括：

- 岗位必要性偏弱。
- 离职影响偏弱。
- 同岗胜任力偏弱。
- 上手速度偏弱。
- 优化意识偏弱。
- 工作闭环偏弱。
- 抗压表现偏弱。
- 主管主观评级与20题重算评级差异较大。
- 自动负荷或员工自评与主管认定差异较大。

## 十一、可执行生成流程

1. 识别当前范围：读取当前目录工作摸排表和主管评价表中出现的部门；单部门输出 `部门名-`，多部门非全量输出 `部分部门-`，全量输出 `全部部门-` 或沿用脚本既有命名。
2. 构造有效员工底表：优先用 ERP `职能部门人员台账` 的机关员工范围；单部门/部分部门时用当前输入表出现人员收紧；剔除公司领导、待岗、中层或副总师等不纳入口径人员。
3. 读取工作摸排表，按 `部门 + 姓名` 取最新填写记录，计算已填、未填、填报率。
4. 读取主管评价表，按 `成员部门 + 成员姓名` 取有效评价，计算已评、未评、完成率；主管评价文件缺失时已评为 0。
5. 引用主管评价20题重算结果，形成 `20题规则重算评级`；主管评价样本为 0 时写 `未评级`，不生成等级矩阵。
6. 生成工作负荷与岗位状态、ABCD等级分布、一致性、不一致名单、岗位必要性反差、低负荷交叉、特殊人群和综合建议。
7. 输出 MD 和 PDF，PDF 表头浅蓝、宽表说明列加宽。

## 十二、关键字段来源

| 字段 | 来源 | 说明 |
| --- | --- | --- |
| 当前范围员工 | ERP人员台账 + 当前输入表 | 当前部门/部分部门时按输入文件收紧。 |
| 工作摸排是否完成 | 工作摸排表 | 同姓名多记录取最新填写日期。 |
| 主管评价是否完成 | 主管评价表 | 同姓名多记录可按最新或逐条，完成情况按人员去重。 |
| 自动生成评级 | 主管评价表 | 只用于与主管主观评级比较。 |
| 主管主观评级 | 主管评价表 | 人工评级，前后对比排序基准。 |
| 20题规则重算评级 | 主管评价20题重算报告 | 后续分析主等级。 |
| 薪资档次 | 主管评价表 `档级` | 仅用于主管侧展示。 |
| 主管定档薪资 | 主管评价表 `定档薪酬` | 直接取原值，不按等级重算。 |

## 十三、0 样本与缺失输入处理

1. 工作摸排表缺失时，工作摸排完成率为 0，未填报人员按当前员工底表列示。
2. 主管评价表缺失或当前范围主管评价样本为 0 时，不生成主管评级结构、不一致名单、基础分级矩阵；写 `当前范围未取得主管评价样本`。
3. 某类异常人员为 0 时，小节写 `当前未发现`，不得生成固定名单或建议。
4. 当前范围员工底表为空时，报告仅输出统计口径和缺失说明，不进行等级、负荷和异常分析。

## 十四、薪酬区间一致性口径

1. 本报告不使用 `主管评级对应公司等级` 字段。
2. 前后对比应使用 `主管主观评级` 与 `20题规则重算评级`。
3. 主管侧薪资档次和定档薪资直接读取 `档级`、`定档薪酬`。
4. 规则重算侧薪资按 20题重算等级映射中位数。
5. 不一致名单列顺序：部门、姓名、20题重算评级、主管主观评级、差异判断、主管评级薪资区间最小值、主管评级薪资区间最大值。

## 十五、章节生成细则

| 章节 | 有样本时 | 无样本时 |
| --- | --- | --- |
| 未填报工作摸排人员名单 | 按部门列人数和姓名 | 写无或按底表全员未填报 |
| 各部门员工评价完成情况 | 按部门列应评、已评、未评、完成率 | 已评 0，未评列当前范围人员 |
| 工作负荷与岗位状态 | 展示员工自评、自动负荷、主管负荷 | 无主管时只展示可用的自评/自动负荷 |
| ABCD等级分布 | 自动、主管、重算三类评级 | 无主管时跳过主管和重算分析 |
| 取低档异常 | 列 B类及以上异常、C类异常 | 无异常写当前未发现 |
| 特殊人群 | E类、毕业两年内、试用或培养期 | 无人员写当前未发现 |

## 十六、生成后复核方法

1. 核对应填人数 = 已填 + 未填，评价应评人数 = 已评 + 未评。
2. 检查未填报和未评价名单显示具体姓名。
3. 检查报告中没有 `主管评级对应公司等级` 分析。
4. 检查所有等级字段与主管评价20题重算报告一致。
5. 检查综合报告引用的填报率、未填报、未评价和等级对比与本报告一致。
