# 重点问题反馈结论分析规则

## 一、规则定位

本文件用于指导生成、检查和修改 `重点问题反馈结论分析`。本报告应贴合原版参考文件的章节和表达，不能只做简单汇总。

## 二、必须读取的输入

- `重点问题反馈汇总`
- `重点问题反馈原始分析方法*.md`
- `职能部门工作量摸排表*.xlsx`
- `部门成员工作主管评定*.xlsx`
- `机关员工主管评价20题重算报告`

## 三、必须包含的章节

1. 总体情况
2. 员工主观想法情况
3. 员工最应取消、合并或简化的工作
4. 员工最值得推动但尚未落地的优化建议
5. 员工对问卷本身的不妥反馈和优化建议
6. 主管评价反馈情况
7. 家庭照护与“妈妈岗”相关情况
8. 前置条件填写情况
9. “妈妈岗”意向情况
10. 前置条件与“妈妈岗”意向交叉判断
11. 岗位优化或调整机会下的工作状态倾向
12. 综合判断

## 四、必须保留的表达模块

在有实际反馈时，应保留类似表达：

- `典型部门与反馈方向如下：`
- `其中较为典型的建议包括：`
- `具体较集中的反馈有：`
- `主要集中在以下方向：`
- `后续建议：`

如果对应统计为 0 人，应改写为 `当前未形成集中反馈`，不得继续生成固定意见列表。

## 五、矛盾控制

1. 若明确“不妥”意见为 0 人，不得写“主要意见包括”。
2. 若妈妈岗明确意向为 0 人，不得写“少量员工表现出强烈意向或有意向”。
3. 选项统计中 `无意向` 和 `没有意向` 不得被解释为有意向。
4. 家庭照护压力只能作为观察信号，不得直接推出妈妈岗需求规模。

## 六、验收清单

- 文字结论与统计表人数一致。
- 原版关键表达模块没有无故丢失。
- 0 人章节没有矛盾描述。
- 当前范围描述准确。

## 七、逐项分析标准

| 分析项 | 有数据时写法 | 0 人或无数据时写法 |
| --- | --- | --- |
| 取消、合并、简化建议 | 写明典型部门、典型事项、集中方向 | 写当前未形成集中反馈，不列固定问题 |
| 尚未落地优化建议 | 写明较典型建议和可推进方向 | 写当前未形成明确优化建议 |
| 问卷不妥反馈 | 写明人数、主要意见、具体问题 | 写明确不妥意见为 0，不再列主要意见 |
| 问卷优化建议 | 写明人数、集中建议 | 若为 0，写暂无明确优化建议 |
| 主管补充反馈 | 写明典型评价方向和人员 | 无反馈则写当前未形成补充反馈 |
| 妈妈岗意向 | 写明 A/B 明确意向人数和人员 | 明确意向 0 人时，不写少量人员有意向 |
| 家庭照护 | 写明照护压力结构 | 只作观察，不推出政策需求规模 |
| 岗位优化倾向 | 按等级分析倾向 | 无数据时写未形成可判断样本 |

## 八、可执行生成流程

1. 先读取 `重点问题反馈汇总`，以其中统计表和名单作为事实来源。
2. 读取 `重点问题反馈原始分析方法*.md`，保持原始分析方法中的分类逻辑和章节顺序。
3. 按当前范围识别单部门、部分部门或全量，所有结论用对应范围描述。
4. 逐章节生成结论：先给人数和范围，再给典型方向，再给后续建议。
5. 每个章节先判断人数是否为 0；为 0 时输出无数据写法，不得套用有数据模板。
6. 妈妈岗意向只统计 A/B 明确意向，C/D/未填写不得解释为潜在意向。
7. 岗位优化倾向必须结合等级维度，不能只按选项总数写结论。

## 九、结论句式模板

| 场景 | 推荐句式 |
| --- | --- |
| 有集中反馈 | `典型部门与反馈方向如下：` 后列部门和方向。 |
| 有典型建议 | `其中较为典型的建议包括：` 后列事项。 |
| 有具体问题 | `具体较集中的反馈有：` 后列问题。 |
| 有多个方向 | `主要集中在以下方向：` 后列方向。 |
| 无集中反馈 | `当前未形成集中反馈。` |
| 无明确意向 | `当前未发现明确有意向人员。` |
| 样本不足 | `当前样本量较小，相关结论宜作为观察信号。` |

## 十、矛盾校验规则

1. 如果某节写 `共有0人`，同节后文不得出现 `主要意见包括`、`典型建议包括`。
2. 如果妈妈岗 A/B 人数为 0，不得出现 `少量员工表现出强烈意向或有意向`。
3. 如果家庭照护压力较多但妈妈岗意向为 0，只能写 `存在照护背景，但未直接转化为明确妈妈岗意向`。
4. 如果问卷不妥反馈为 0、优化建议大于 0，两者必须分开写，不能混成“不妥反馈主要包括”。

## 十一、输出结构细则

1. `总体情况`：说明有效反馈人数、反馈类型覆盖情况。
2. `员工主观想法情况`：说明有想法人数、典型部门、主要方向。
3. `取消/合并/简化`：按事项归纳，不改写员工原意。
4. `尚未落地优化建议`：突出可推进事项。
5. `问卷反馈`：不妥反馈和优化建议分开。
6. `主管反馈`：同一主管对同一员工反馈合并后概括。
7. `家庭照护与妈妈岗`：先统计家庭照护，再判断妈妈岗意向。
8. `岗位优化倾向`：按等级分布判断倾向，不直接形成岗位调整结论。
9. `综合判断`：只提出后续访谈、政策观察、问卷优化等建议。

## 十二、生成后复核方法

1. 每个结论人数必须能在反馈汇总表中找到对应统计。
2. 原版关键表达模块在有数据时必须出现。
3. 0 人章节不得出现固定意见列表。
4. 结论中的人员名单不得超出当前范围。
