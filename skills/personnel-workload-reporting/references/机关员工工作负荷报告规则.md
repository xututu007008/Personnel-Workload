# 机关员工工作负荷报告规则

## 一、规则定位

本文件用于指导生成、检查和修改 `机关员工工作负荷报告`。本报告用于比较员工自评、自动规则认定和主管主观认定之间的工作负荷差异。

## 二、必须读取的输入

- `职能部门工作量摸排表*.xlsx`
- `部门成员工作主管评定*.xlsx`
- `机关员工工作负荷认定规则*.md`
- `机关员工主管评价20题重算报告`

## 三、三类负荷定义

| 类型 | 来源 | 用途 |
| --- | --- | --- |
| 员工自评负荷 | 工作摸排表员工填报 | 反映个人主观感受 |
| 自动工作负荷 | 按工作负荷认定规则计算 | 形成统一口径 |
| 主管主观负荷 | 主管评价表 | 反映主管管理视角 |

三者必须同时保留，不得互相覆盖。

## 四、必须包含的章节

1. 报告口径
2. 整体结果
3. 三类工作负荷分布
4. 工时折算统计
5. 样本分布
6. 工时异常情况
7. 差异情况
8. 员工自评与主管主观认定差异
9. 自动工作负荷与主管主观认定差异
10. 员工自评与主管认定差异较大样本
11. 自动负荷与主管认定差异较大样本
12. 人员明细

`差异情况` 下的两个“大样本”小节必须保留。

## 五、差异判断

1. 员工自评明显高于主管认定：员工认为忙，但主管认为一般或不忙。
2. 员工自评明显低于主管认定：员工认为不忙，但主管认为忙。
3. 自动负荷高于主管认定：规则识别工作量较高，但主管未认定为忙。
4. 自动负荷低于主管认定：规则识别不高，但主管认为忙。

差异人员名单应包含姓名、部门、三类负荷、等级和差异说明。

## 六、等级口径

所有人员等级均使用 `机关员工主管评价20题重算报告` 的综合重算评级。不得使用表内自动评级替代。

## 七、表格与 PDF

1. 表头浅蓝底色。
2. 负荷分布表列宽均衡。
3. 差异样本表中 `差异说明` 列加宽。
4. 长文本反馈合并在同一单元格内，用分号或换行区分。
5. 行高要比默认 Markdown 表格更舒展。

## 八、验收清单

- 三类负荷均存在。
- 差异样本两个小节不缺失。
- 低负荷人员与综合报告名单一致。
- 当前范围为单部门或部分部门时，文字描述同步收紧。

## 九、逐项认定标准

| 项目 | 读取来源 | 输出标准 |
| --- | --- | --- |
| 员工自评总体工作负荷 | 工作摸排表 | 原样归类展示，不得用主管评价覆盖。 |
| 自动工作负荷 | 工作负荷认定规则 | 按规则统一判定，必须能追溯到规则条件。 |
| 主管主观工作负荷 | 主管评价表忙闲题 | 主管视角单独统计。 |
| 工时折算 | 工作摸排填报工时或频次 | 分部门、分人员统计，异常值保留。 |
| 工时异常 | 折算工时过高或过低 | 进入异常名单并说明原因。 |
| 员工自评与主管差异 | 自评负荷 vs 主管负荷 | 差异较大样本必须列人员。 |
| 自动负荷与主管差异 | 自动负荷 vs 主管负荷 | 差异较大样本必须列人员。 |

## 十、差异较大样本列示标准

| 列名 | 要求 |
| --- | --- |
| 部门 | 当前归属部门 |
| 姓名 | 员工姓名 |
| 按20题计算等级 | 使用重算等级 |
| 员工自评负荷 | 原始自评 |
| 自动工作负荷 | 规则认定 |
| 主管主观负荷 | 主管认定 |
| 差异说明 | 说明高低方向和复核原因 |

## 十一、可执行生成流程

1. 读取 `职能部门工作量摸排表*.xlsx`，以 `部门 + 姓名` 为人员主键，保留最新填写记录。
2. 读取 `部门成员工作主管评定*.xlsx`，以 `成员部门 + 成员姓名` 为主键读取主管忙闲题；若主管评价文件缺失或当前部门无样本，主管主观负荷写空或 `无主管评价`，报告不报错。
3. 读取 `机关员工工作负荷认定规则*.md`，按规则从工作事项、频次、工时、复杂度等字段计算自动工作负荷。
4. 从主管评价重算报告读取 `按20题计算等级`；若无主管评价样本，则等级写 `未评级`，不得用自动评级替代。
5. 生成三类负荷字段：员工自评负荷、自动工作负荷、主管主观负荷。
6. 计算三类负荷分布和交叉差异；差异大样本分别输出员工自评 vs 主管、自动负荷 vs 主管两张表。
7. 输出人员明细，保证每个当前范围员工都有一行。

## 十二、负荷层级归一化

| 归一层级 | 可识别文本 |
| --- | --- |
| 超负荷 | 超负荷、严重超负荷、长期超负荷 |
| 满负荷 | 满负荷、较忙、繁忙、工作饱和 |
| 基本饱和 | 基本饱和、适中、正常 |
| 非饱和 | 非饱和、不忙、空闲、欠饱和 |
| 无法判断 | 空白、未填写、无明确描述 |

差异比较按 `超负荷 > 满负荷 > 基本饱和 > 非饱和 > 无法判断` 排序；无法判断不参与高低差异结论，只进入缺失说明。

## 十三、差异判定细则

1. 员工自评比主管认定高 2 档及以上，列入 `员工自评高于主管认定较大样本`。
2. 员工自评比主管认定低 2 档及以上，列入 `员工自评低于主管认定较大样本`。
3. 自动负荷比主管认定高 2 档及以上，列入 `自动负荷高于主管认定较大样本`。
4. 自动负荷比主管认定低 2 档及以上，列入 `自动负荷低于主管认定较大样本`。
5. 若主管认定为空，不进入差异较大样本，但应在完成情况或说明中提示主管样本缺失。

## 十四、章节内容模板

1. `报告口径`：说明三类负荷来源、当前范围和主管评价缺失处理。
2. `整体结果`：列当前范围人数、可计算自动负荷人数、存在主管认定人数。
3. `三类工作负荷分布`：分别列员工自评、自动负荷、主管认定分布。
4. `差异情况`：先给总体差异人数，再展开两个必保小节。
5. `人员明细`：部门、姓名、等级、三类负荷、工时折算、差异判断。

## 十五、空样本写法

1. 工作摸排人员为 0 时，写 `当前范围未取得工作摸排样本`，人员明细只保留表头。
2. 主管评价人员为 0 时，差异小节写 `当前范围未取得主管评价样本，暂不开展员工自评/自动负荷与主管认定差异比较`。
3. 差异较大样本为 0 时，写 `当前未发现差异较大样本`，不得生成空白解释段。

## 十六、生成后复核方法

1. 核对人员明细行数等于当前工作摸排有效员工数。
2. 核对三类负荷分布合计分别等于其有效样本数。
3. 抽查差异较大人员，确认差异方向与层级顺序一致。
4. 综合报告中的低负荷信号必须来自本报告差异和主管负荷结论。
