# 重点问题反馈汇总规则

## 一、规则定位

本文件用于指导生成、检查和修改 `重点问题反馈汇总`。本报告汇总员工开放反馈、主管补充反馈、家庭照护、妈妈岗意向、岗位优化倾向和问卷本身反馈。

## 二、必须读取的输入

- `职能部门工作量摸排表*.xlsx`
- `部门成员工作主管评定*.xlsx`
- `重点问题反馈原始分析方法*.md`
- `机关员工主管评价20题重算报告`

## 三、必须包含的章节

1. 汇总
2. 分层关注名单
3. 员工主观想法名单
4. 员工认为最应取消、合并或简化的工作
5. 员工认为最值得推动但尚未落地的优化建议
6. 员工认为问卷中存在的不妥情况
7. 员工对问卷需要优化的问题和建议
8. 主管评价补充反馈
9. 妈妈岗有意向人员名单
10. 家庭照护与岗位优化倾向统计
11. 岗位优化或调整机会下的工作状态倾向

## 四、合并规则

1. 同一员工针对同一控件的多条意见合并到一个单元格。
2. 同一主管针对同一员工的评价合并到一个单元格。
3. 不同控件的意见可以分列展示。

## 五、家庭照护统计

家庭照护章节必须输出统计表：

| 问题 | 选项 | 人数 |
| --- | --- | --- |

统计字段包括婚育情况、重家庭照护责任、12岁以下子女照护、配偶是否同公司且无法居家、妈妈岗意向、岗位优化倾向等。

## 六、岗位优化倾向统计

`岗位优化或调整机会下的工作状态倾向` 必须增加等级维度：

| 等级 | 工作状态倾向 | 人数 |
| --- | --- | --- |

等级列和人数列较窄，工作状态倾向列较宽。

## 七、验收清单

- 妈妈岗有意向名单不含无意向人员。
- 家庭照护统计存在。
- 同人同控件反馈已合并。
- 等级来自 20 题重算结果。

## 八、逐项统计标准

| 控件/问题 | 统计方式 | 输出位置 |
| --- | --- | --- |
| 婚育情况 | 按选项计数 | 家庭照护统计 |
| 是否承担较重家庭照护责任 | 按选项计数，未填写单独列 | 家庭照护统计 |
| 12岁以下子女照护 | 按选项计数，未填写单独列 | 家庭照护统计 |
| 配偶是否同公司且无法居家 | 按选项计数，未填写单独列 | 家庭照护统计 |
| 妈妈岗意向 | 按选项计数，并另列有意向人员 | 妈妈岗章节 |
| 岗位优化后的工作状态倾向 | 按等级 + 选项交叉计数 | 岗位优化倾向章节 |
| 员工开放意见 | 按人员聚合 | 员工主观想法名单 |
| 主管补充反馈 | 按员工聚合 | 主管评价补充反馈 |

## 九、文本聚合标准

1. 同一人员同一字段多条文本，用 `；` 合并。
2. 不同字段不得混合到同一列。
3. 合并后仍为空的人员，不进入该字段名单。
4. 文本中明显重复内容可去重，但不得改写原意。

## 十、可执行生成流程

1. 读取工作摸排表，提取员工开放反馈、家庭照护、妈妈岗意向、岗位优化倾向等字段。
2. 读取主管评价表，提取主管补充反馈、主管对问卷或员工管理的补充意见；主管评价文件缺失时主管反馈章节写 `当前未取得主管评价样本`。
3. 读取主管评价20题重算报告，按姓名补入等级；无主管样本时等级写 `未评级`。
4. 按 `部门 + 姓名 + 控件字段` 聚合文本，同一人同一控件多条内容合并到一个单元格。
5. 生成家庭照护统计表：每个问题按选项计数，未填写单独列示。
6. 生成妈妈岗统计和有意向名单：只把 A/B 明确正向选项纳入名单。
7. 生成岗位优化倾向等级交叉表：等级、工作状态倾向、人数。
8. 输出汇总表和人员名单，保证所有名单均为当前范围人员。

## 十一、字段识别标准

| 类别 | 常见字段关键词 | 处理 |
| --- | --- | --- |
| 取消/合并/简化工作 | 取消、合并、简化、最应取消 | 非空有效文本进入员工主观想法。 |
| 尚未落地优化建议 | 值得推动、尚未落地、优化建议 | 非空有效文本进入优化建议。 |
| 问卷不妥反馈 | 问卷、不妥、不安、问题 | 非空有效文本进入问卷反馈。 |
| 问卷优化建议 | 问卷优化、需要优化、建议 | 非空有效文本进入问卷优化。 |
| 主管补充反馈 | 补充反馈、主管评价、其他意见 | 按主管和员工聚合。 |
| 妈妈岗意向 | 妈妈岗、照顾政策、意向 | A/B 为有意向，C/D/未填写为无明确意向。 |
| 岗位优化倾向 | 岗位优化、调整机会、工作状态 | 按等级 + 选项统计。 |

## 十二、无效文本与去重

以下内容视为无效，不进入有想法名单：空白、`无`、`没有`、`暂无`、`无意见`、`不涉及`、`无不妥`、`未填写`、`nan`。

去重时只去掉完全重复或清洗后完全相同的文本；不得把相近但含义不同的反馈合并改写。

## 十三、妈妈岗意向细则

| 原始选项 | 是否进入有意向名单 |
| --- | --- |
| A.有强烈意向 | 是 |
| B.有意向，但是不迫切，需考虑 | 是 |
| C.无意向 / C.没有意向 | 否 |
| D.无意向，但是可能会动态变化 | 否 |
| 未填写 / 空白 | 否 |

家庭照护压力、已婚已育、12岁以下子女等只能作为背景统计，不能自动推导为妈妈岗有意向。

## 十四、空样本写法

1. 员工开放反馈为 0 时，对应名单写 `当前未形成有效反馈`。
2. 妈妈岗明确意向为 0 时，名单写 `当前未发现明确有意向人员`。
3. 家庭照护统计中某选项为 0 可以不展示；但问题本身应保留。
4. 岗位优化倾向无数据时，写 `当前未形成可统计样本`。

## 十五、生成后复核方法

1. 抽查妈妈岗名单，确认不含 C/D/未填写人员。
2. 抽查同一员工多条反馈，确认已合并到同一单元格。
3. 家庭照护统计各问题选项人数合计应等于当前有效样本数或包含未填写后等于样本数。
4. 岗位优化倾向等级交叉人数应等于该题有效填写人数。
