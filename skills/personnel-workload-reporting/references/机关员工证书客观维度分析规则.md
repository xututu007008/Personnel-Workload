# 机关员工证书客观维度分析规则

## 一、规则定位

本文件用于指导生成、检查和修改 `机关员工证书客观维度分析`。本报告从职称证书、资格证书、荣誉证书、人力资源复核、年终总结匹配等客观信息辅助观察员工等级合理性。

## 二、必须读取的输入

1. `职能部门工作量摸排表*.xlsx`
   - 部门、姓名、工作年限、职称证书、资格证书、荣誉证书。
2. `ERP使用情况.xlsx` 的 `职能部门人员台账`
   - `优秀情况（文本化）`
   - `安全先进个人情况（文本化）`
3. `人员证书复核` 图片或等效资料
   - 用于修正无证书人员名单。
4. `年终总结*` 文件
   - 用于识别年终总结未匹配名单。
5. `机关员工主管评价20题重算报告`
   - 提供 `按20题计算等级`。

## 三、证书合并口径

| 字段 | 合并规则 |
| --- | --- |
| 职称证书 | 来自摸排表，原样保留并清理空白 |
| 资格证书 | 来自摸排表和 HR 复核补充 |
| 荣誉证书 | 摸排表荣誉 + 主管/员工补充 + ERP台账优秀情况 + ERP台账安全先进个人情况 |

多个证书或荣誉用换行展示，不要压成一串。

## 四、无证书判断

1. 职称证书、资格证书、荣誉证书均为空时，初判为 `无任何证书`。
2. HR 复核资料显示已有证书的人员，必须从 `无任何证书` 中去除。
3. HR 复核资料不涉及当前部门人员时，不显示 HR 复核说明。
4. 工作满 2 年仍无任何证书且等级较高的人员，进入复核关注。

## 五、必须包含的章节

1. 分析口径
2. 整体结论
3. 按等级部门汇总
4. 年终总结未匹配名单
5. 人力资源复核明细
6. 按等级人员证书列表
7. 无任何证书人员按主管评价等级分布
8. 工作满2年仍无任何证书人员名单
9. 工作未满2年且无任何证书人员名单

`按等级部门汇总` 必须同时有等级和部门维度，不能只按等级汇总。

## 六、人员明细表列顺序

| 列名 | 说明 |
| --- | --- |
| 部门 | 当前部门 |
| 姓名 | 员工 |
| 工作年限 | 保留 1 位小数或原脚本口径 |
| 按20题计算等级 | 主管评价重算等级 |
| 按20题计算等级对应薪资中位数 | 按重算等级映射 |
| 主管评价等级 | 主管主观评级 |
| 主管评价等级对应薪资中位数 | 按主管评级映射 |
| 职称证书 | 较窄长文本列 |
| 资格证书 | 较窄长文本列 |
| 荣誉证书 | 最宽长文本列 |

## 七、PDF 排版

1. 表头浅蓝底色。
2. `荣誉证书` 列必须显著加宽。
3. 证书列使用换行，避免竖排。
4. 宽表左右边距保留，不要贴边。

## 八、验收清单

- HR 复核有证书人员不在无证书名单。
- ERP 台账优秀和安全先进信息进入荣誉证书。
- 年终总结未匹配名单存在。
- 等级与主管评价重算报告一致。

## 九、逐项字段标准

| 字段 | 来源 | 处理标准 |
| --- | --- | --- |
| 工作年限 | 摸排表或人员台账 | 保留数值，空值不得自动补 0。 |
| 职称证书 | 摸排表 | 空、无、暂无视为空；有效文本保留。 |
| 资格证书 | 摸排表 + HR复核 | HR 复核补充的证书必须写入。 |
| 荣誉证书 | 摸排表 + ERP台账 | 合并优秀员工、安全先进个人等文本化荣誉。 |
| 年终总结匹配 | 年终总结文件 | 未匹配人员单独列示。 |
| 按20题计算等级 | 重算报告 | 不得使用表内自动评级替代。 |
| 主管评价等级 | 主管评价表 | 仅作对照展示。 |

## 十、无证书人员复核标准

1. 初筛无证书：职称、资格、荣誉均为空。
2. HR 复核：若 HR 截图或复核材料显示其有证书，则从无证书名单移除。
3. 当前范围判断：HR 复核材料中不属于当前部门范围的人员，不在报告中展开说明。
4. 下调复核：工作满 2 年、复核后仍无证书、且等级较高的人员，进入建议复核范围。

## 十一、可执行生成流程

1. 读取当前范围人员：优先从工作摸排表获得 `部门、姓名、工作年限、职称证书、资格证书、荣誉证书`；若工作摸排表缺失，则从主管评价或 ERP 台账补人员，但工作年限为空。
2. 读取主管评价20题重算报告，按姓名补入 `按20题计算等级`、重算等级薪资中位数、主管评价等级、主管评价等级薪资中位数。
3. 读取 `ERP使用情况.xlsx` 的 `职能部门人员台账`，按姓名补入 `优秀情况（文本化）` 和 `安全先进个人情况（文本化）`，合并到荣誉证书。
4. 读取当前目录 `年终总结*` 文件，按姓名匹配有效职称证书、有效资格证书、其他荣誉证书；未匹配人员进入 `年终总结未匹配名单`。
5. 读取人力资源 `人员证书复核` 图片或等效资料；只处理当前范围内人员。复核显示有证书者从无证书名单移除，并在资格证书或复核说明中体现。
6. 生成按等级部门汇总：必须同时包含等级、部门、人数和人员名单，不得只按等级汇总。
7. 生成按等级人员证书列表和无证书人员分布。

## 十二、证书文本清洗与合并

1. 以下值视为空：空白、`无`、`暂无`、`没有`、`nan`、`/`、`-`。
2. 同一来源多个证书用换行分隔；跨来源合并时去重但不改写原文。
3. 荣誉证书合并顺序：工作摸排荣誉、年终总结荣誉、ERP台账优秀情况、ERP台账安全先进个人情况。
4. HR 复核补充证书应保留复核来源标识，例如 `HR复核：安全A证`。
5. 若 HR 复核确认仍无证书，证书列保持空，但复核明细写 `无`。

## 十三、等级与薪资映射

| 字段 | 口径 |
| --- | --- |
| 按20题计算等级 | 来自主管评价20题重算报告；主管样本为 0 时写 `未评级`。 |
| 按20题计算等级对应薪资中位数 | 按等级 S/A/B/C/D/E/F 映射；未评级为空。 |
| 主管评价等级 | 来自主管主观评级；缺失写空。 |
| 主管评价等级对应薪资中位数 | 按主管评价等级映射；缺失为空。 |

不得用证书情况反推等级。

## 十四、空样本写法

1. 当前范围人员为 0 时，整体结论写 `当前范围未取得有效人员样本`。
2. HR 复核资料不涉及当前范围人员时，不显示 `人力资源复核明细` 章节。
3. 年终总结文件缺失时，保留 `年终总结未匹配名单` 章节并说明 `当前目录未提供年终总结文件`。
4. 无任何证书人员为 0 时，写 `当前未发现无任何证书人员`，不得继续生成下调复核建议。

## 十五、输出表格要求

1. `按等级部门汇总` 列：等级、部门、人数、人员名单。
2. `按等级人员证书列表` 列：部门、姓名、工作年限、按20题计算等级、按20题计算等级对应薪资中位数、主管评价等级、主管评价等级对应薪资中位数、职称证书、资格证书、荣誉证书。
3. `无任何证书人员按主管评价等级分布`：主管评价等级、人数、人员名单；主管评价缺失时列 `未评级`。
4. 荣誉证书列必须最宽，允许多行换行。

## 十六、生成后复核方法

1. 抽查 HR 复核显示有证书人员，确认不在无证书名单。
2. 抽查 ERP 台账优秀/安全先进人员，确认进入荣誉证书列。
3. 核对按等级部门汇总人数合计等于当前范围人员数。
4. 核对综合报告证书维度引用的无证书名单与本报告复核后名单一致。
